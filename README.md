## 📖 Использование

### Основной процесс сортировки

1. **Выберите исходную папку** - директорию с файлами для сортировки
2. **Выберите целевую папку** - где будут созданы папки категорий
3. **Настройте параметры:**
   - Обработка подпапок (рекурсивно)
   - Игнорирование скрытых файлов
   - Режим обработки дубликатов
4. **Нажмите "Предпросмотр"** - увидьте результат до применения
5. **Нажмите "Запустить сортировку"** - примените изменения

### Настройка категорий

1. Откройте **"Управление категориями"**
2. Создайте новую категорию или отредактируйте существующую
3. Настройте правила:
   - **По расширениям**: `.jpg, .png, .gif`
   - **По базе данных имен**: список конкретных имен файлов/папок
   - **По контроллеру**: интеграция со Steam и другими лаунчерами

### Интеграция со Steam

Для автоматического распознавания Steam игр:

1. Создайте/откройте категорию "Steam игры"
2. Включите опцию **"Использовать контроллер"**
3. Выберите тип **"Steam"**
4. Нажмите **"Сканировать директорию"**
5. Укажите путь к `Steam/steamapps/common/`
6. Сохраните изменения

Контроллер автоматически:
- Добавит названия всех игр в базу данных
- Будет проверять `steam_appid.txt` для точного распознавания
- Распознает Steam ярлыки по AppID
